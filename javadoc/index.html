Custom_Crafter

Welcome to Custom_Crafter official document top page !
List of elements

  • event: Custom_Crafter events

  • interfaces: CMatter, CRecipe and more interfaces

  • objects: Elements of CMatter and CRecipe

  • matter: Default CMatter implementation classes

  • recipe: Default CRecipe implementation classes and CRecipeFilter implementations

  • result: Default ResultSupplier implementation class

  • util: public utilities

This product includes software developed by the IMT Atlantique.

  • kotlin-stdlib: Kotlin std library

  • paper-api: PaperMC API

  • choco-solver: As a combination solver for Shapeless recipes

  • FoliaLib: For compatibility with the Folia platform
